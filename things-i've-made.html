Pipelined Processor.

Computer Organization and Architecture , McGill university

I've taken a Computer organization and Architecture course at the McGill University. The project for this course was to develop a standard, five-stage, 32-bit pipelined MIPS processor. It contains write-back, direct-mapped instruction cache and data cache, Hazard Detection Unit, Forward Unit, and Branch Prediction Unit. It was mainly developed by VHDL and simulated in ModelSim.

Android-Based Mecanum Wheel Robot Car.

The Hong Kong Polytechnic University

This Mecanum Wheel Robot car is based on an Android application that enables user to control its speed and direction. However, we set a constant speed for this car, which requires each wheel has exactly the same speed. The PID control program can make sure the car could always maintain the required speed on any floor regardless of the friction. As one of the field buses, CAN bus is used to communicate between main controller and 2 motor controllers. Bluetooth HC-05 is used to communicate between phone and main controller.
